OGGETTI

Gli oggetti in js si possono claddificare in base allo scopo e modo di creazione.

1. Oggetti Letterali

let oggetto = { nome: 'Mario', eta: 30 };

2. Oggetti Creati da Costruttori

Questi oggetti vengono creati usando funzioni costruttrici insieme alla parola chiave new. JavaScript offre diversi costruttori integrati come Object, Array, Function, Date, RegExp, Error, e altri.

let data = new Date(); let array = new Array(1, 2, 3);

3. Istanze di Classi

Con l'introduzione delle classi in ES6, puoi definire classi e creare istanze di queste classi usando new. Le classi possono essere viste come un "zucchero sintattico" sopra le funzioni costruttrici per facilitare la creazione di oggetti e l'ereditarietà.

class Persona { constructor(nome) { this.nome = nome; } } let persona = new Persona('Mario');

4. Oggetti Singleton tramite new Object()

5. Oggetti Function

Le funzioni in JavaScript sono oggetti di prima classe, il che significa che possono avere proprietà e metodi, essere assegnate a variabili, passate come argomenti o restituite da altre funzioni.

function saluta() { console.log('Ciao'); } saluta.messaggio = 'Benvenuto';

6. Oggetti Array

Gli array in JavaScript sono oggetti specializzati per rappresentare collezioni di dati ordinati e accessibili tramite indice.

let numeri = [1, 2, 3];

7. Oggetti Globali

Sono oggetti che esistono automaticamente nell'ambiente di esecuzione globale. Esempi includono oggetti globali come Math, JSON, e altri specifici dell'ambiente come window nel browser e global in Node.js.

8. Oggetti Prototipo

Ogni funzione in JavaScript ha una proprietà prototype che è un oggetto. Questo oggetto prototype viene usato quando crei istanze di quella funzione tramite new. Gli oggetti creati erediteranno da questo oggetto prototype

9. Oggetti creati con Object.create()

Object.create(proto[, propertiesObject]) crea un nuovo oggetto con l'oggetto prototipo specificato e, opzionalmente, proprietà specificate.

let oggetto = Object.create(null); // crea un oggetto senza prototipo

Differenze

sia gli oggetti letterali che quelli creati tramite funzioni costruttrici ereditano il comportamento di base dal prototipo Object.prototype, inclusa la disponibilità del metodo toString(). La differenza principale tra i due modi di creare oggetti non riguarda l'accesso ai metodi come toString(), ma piuttosto come si struttura il codice, si gestisce l'ereditarietà e si realizzano istanze multiple con comportamenti condivisi.

map()

prende ogni elemento dell'array di partenza gli applica la callback e costruisce man mano il nuovo array con tutti i risultati ottenuti.

esegue una funzione per ogni elemento di un array

restituisce un nuovo array

const ages = people.map(person => person.age); console.log(ages);

reduce()

esegue un'operazione su ogni elemento mettendo il risultato in un accumulatore che può avere un valore iniziale

const numbers = [1, 2, 3, 4, 5]; // Utilizzo di reduce() per sommare tutti gli elementi const sum = numbers.reduce((accumulator, current) => { return accumulator + current; }, 0); // 0 è il valore iniziale dell'accumulatore console.log(sum);

reduce() per creare un oggetto

Quando fai il return newObj dentro la funzione di callback di reduce(), non stai creando un nuovo array. Invece, stai passando l'oggetto newObj che hai modificato al prossimo ciclo di iterazione di reduce(). Alla fine di tutte le iterazioni, reduce() ritorna il valore finale di newObj, che è un singolo oggetto contenente tutte le proprietà che hai aggiunto o modificato durante le iterazioni.

let risultato = [ {nome: "Zoff", ruolo: "portiere"}, {nome: "Cabrini", ruolo: "difensore"}, {nome: "Tardelli", ruolo: "centrale"}, {nome: "Rossi", ruolo: "attaccante"}, ]; let team = risultato.reduce((newObj, el) => { newObj[el.ruolo] = el.nome; return newObj; }, {}) console.log(team); /* {portiere: 'Zoff', difensore: 'Cabrini', centrale: 'Tardelli', attaccante: 'Rossi'} */
